# EliteCleaning MVP (Next.js)

## Ejecutar
1) Instalar dependencias:
```bash
npm install
```

2) Iniciar:
```bash
npm run dev
```

Abre: http://localhost:3000

## Rutas
- / (Home)
- /login
- /cart
- /account
- /admin

Admin demo:
- admin@elite.cl / 1234
